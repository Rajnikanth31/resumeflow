# ResumeFlow Project Roadmap

This document outlines the milestones, epics, and timelines for the ResumeFlow open-source project.

## Completed Milestones (v1.1.0)

- **Epic 1: Engineering Quality Gate**: Standardized ESLint rule checks, preloaded font loader, and fixed dark mode flash (FOUC).
- **Epic 2: CI/CD Quality Controls**: Added Husky, lint-staged, and Conventional Commits validation.
- **Epic 3: Resume Builder & Editor Optimization**: Real-time form validations, dual-pane canvas preview layout, dynamic style switchers, accessibility compliance (A11y), and off-screen PDF printing.

## Short-Term Goals (Q3 2026)

- **Epic 4: ATS Audit & Scan Tooling**: Run semantic validation scanning against job descriptions, suggesting keywords to insert.
- **Epic 5: Portfolio Generator**: Build interactive online portfolios using resume drafts.
- **Epic 6: AI Bullet Points Improver**: Suggest action verbs and metrics-driven results.

## Long-Term Vision (Q4 2026+)

- **Epic 10: Multi-Format Import Engine**: Upload DOCX, TXT, or JSON and parse profiles.
- **Epic 12: Teamwork & Project Collaborator**: Share drafts and comments.
- **Epic 13: Template Marketplace**: Community-driven theme plugins.
